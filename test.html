Zaškrtnutím vyberte správné odpovědi na otázky. Správně je vždy právě jedna odpověď, každá otázka je za 2 body, nezodpovězená otázka je za 0 bodů, špatně zodpovězená otázka je za -1 bod.

Otázka č. 1 - Jak vytvoříme SVG dokument knihovny SVG.js?

var draw = SVGJS('drawing').size(300,300)
var draw = document.createSVG('drawing').size(300,300)
var draw = SVG('drawing').size(300,300)
var SVG= draw('drawing').size(300,300)

Otázka č. 2 - Jakým způsobem můžeme vykreslit vodorovnou čáru, která má sílu 3 pixely? Předpokládejme, že již máme vytvořený canvas se jménem draw.

var line = draw.line(0,0,100, 100).stroke({width: 3})
var line = draw.stroke(0,100,100, 100).line({width: 3})
var line = draw.line(0,100,100,100).stroke({width: 3})
var line = draw.line(0,0,100, 100).width('3px')

Otázka č. 4 - Jakým způsobem můžeme získat odkaz na SVG element, který má id "my-element"?

var element = document.getElementById('#my-element')
var element = SVG.select('#my-element')
var element = SVG.getElementById('#my-element')
var element = SVG.get('#my-element')

Otázka č. 5 - Jak byste u SVG objektu uloženého v proměnné rect odstranili stribut stroke?

rect.attr('stroke', null)
rect.stroke(null)
rect.attr('stroke', 0)
rect.removeAttr('stroke')

rect.animate().move(30, 150)
rect.animate({x: 150, y: 30}).move()
